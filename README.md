### Summary

Next.js  + Prisma ORM + mysql 5.7 , CRUD sample

***
### Setup

* npm install

* npx prisma migrate dev --name init

***
### Setup ,etc

* .env , DATABASE_URL : mysql setting required

```
DATABASE_URL = 'mysql://user123:pass123@localhost:3306/dbname'
```


***
### start server
* Start :

yarn dev
